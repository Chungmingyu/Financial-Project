## 설계 내용(아키텍처 등) 및 실제 구현 정도

### **개발 환경**
- **백엔드**: Django, RESTful API
- **프론트엔드**: Vue.js
- **데이터베이스**: SQLite
- **AI**: 머신러닝을 활용한 사용자 맞춤형 추천 및 투자 성향 분석, 생성형 AI 챗봇 활용
- **협업 툴**: Git, Notion, Asana

### **백엔드**
- Django를 사용한 서버 구축
  - 사용자 인증 및 권한 관리
  - 금융 상품 데이터 관리
  - 추천 알고리즘 API 제공
  - 사용자 설문조사 관리
  - 게시판 관리
  - 뉴스 실시간 데이터 관리
  - 지도 실시간 데이터 관리
  - 환율 실시간 데이터 관리

### **프론트엔드**
- Vue.js를 사용한 SPA 구현
  - 사용자 인터페이스(UI) 설계 및 개발
  - 사용자 설문 기능 구현
  - 추천 결과 시각화
  - 금융 상품 UI/UX 구현

### **API 통신**
- RESTful API를 통한 프론트엔드와 백엔드 통신
  - 사용자 데이터 전송 및 수신
  - 추천 결과 전송 및 수신

### **데이터베이스**
- SQLite 사용한 데이터베이스 구축
  - 사용자, 설문지, 금융 상품 등의 테이블 구성
  - 데이터베이스 최적화 및 인덱싱

---

## 데이터베이스 모델링(ERD)

### **주요 테이블**
- **User**: 사용자 정보를 관리.
- **Survey**: 설문 응답 데이터를 저장.
- **Recommendation**: 사용자 맞춤 추천 기록을 관리.
- **moneys**: 금융 관련 데이터 저장
- **boards**: 게시판 데이터 저장

---

## 금융 상품 추천 알고리즘에 대한 기술적 설명
- Random Forest 활용
- 사용자의 투자 성향을 분석하여 적합한 금융 상품을 추천하는 알고리즘 구현
  - 설문 응답 데이터를 기반으로 투자 성향 분석
  - 금융 상품의 금리, 조건 등을 비교하여 최적의 투자 계획 제안
  - AI 모델을 활용하여 추천 정확도 향상

## 서비스 대표 기능들에 대한 설명

### 1. **투자 성향 분석**
- 사용자의 설문 응답 데이터를 기반으로 투자 성향 분석.
- 안정적 투자자와 모험적 투자자로 성향을 분류.

### 2. **맞춤형 금융 상품 추천**
- AI 기반 추천 알고리즘을 통해 적합한 금융 상품 제공.
- 예적금, 주식, 부동산 등 다양한 카테고리의 상품 제안.

### 3. **투자 시뮬레이션**
- 다양한 금융 시나리오를 시각화.
- 금리 변동, 투자 금액에 따른 수익률 예측.

### 4. **부가 서비스**
- 금융 뉴스 제공.
- 설문조사를 통한 맞춤형 사용자 칭호 부여.
- 주변 은행 위치 제공.

### 5. **금융 데이터 소개**
- 예적금 정보 제공
- 실시간 가상화폐 정보 제공
- 주식 정보 제공
- 환율 정보 제공
- 부동산 정보 제공
- 

### 6. **유저 편의 기능 제공**
- 게시판, 댓글 기능 제공
- 회원변경, 설문 기능 제공
- 다른 유저 정보 기능 제공
- 챗봇 기능 제공
- 게시글 좋아요 기능 제공
